# SkillTreeMax

世界最大級のスキルツリーRPGを目指すGitHub Pages向けプロジェクトです。  
`index.html` をそのまま公開する構成で、以下の要素を用意しています。

- 日本語UIのRPGステータス・クエスト・休息アクション
- 系統別に拡張可能な巨大スキルツリー（複数系統 × 階層 × スキル数）
- 今後の拡張を見据えたロードマップ

## GitHub Pagesデプロイ

1. GitHub の Settings → Pages を開く  
2. Branch を `main` に設定し、`/ (root)` を選択  
3. 保存して数分待つと `index.html` が公開されます。
